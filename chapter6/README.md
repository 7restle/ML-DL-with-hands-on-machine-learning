## 6.2 예측
루트 노드에서부터 특성 값을 기준으로 이진 트리 형태로 분할해 내려감  
루트 노드 - 분할 노드 - 리프 노드 순으로 이어짐  
이런 식으로 제작된 각 노드에 대해 지니 불순도를 측정함.
측정된 지니 불순도가 0이면 모든 샘플이 같은 클래스에 속하는 상태, 더 이상 분할할 필요 없음  
지니 불순도가 0이 아니면 분할할 수 있음, 분할은 설정해 둔 depth까지 수행
